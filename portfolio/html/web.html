PROJECT NAME: ePortfolio System

LANGUAGES USED: ASP.NET

MODULE: Web App Development

BRIEF DESCRIPTION:

I created the Parent package, which basically allowed parents to view their own children's portfolios. As per assignment requirements,
they first have to send in a request for each individual portfolio they want to view. The system admin will then verify and check if the
request has been sent by the parent of the child. In the parents' profile, there is a table which shows all the details of the child whose
portfolios they have requested, with the viewing status showing as either 'A' for accepted or 'P' for pending. If the request has been approved
the parent can then click on the child's student ID on the table, and be redirected to theit child's full portfolio. Otherwise, they will only
be shown a partial public version of the child's portfolio.
Parents are also able to send in a message to initiate a converdation with the mentor of their children, and eventually have conversations with
the mentor in a chat-style manner.
